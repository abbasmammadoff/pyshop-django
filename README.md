# PyShop - Django E-commerce Website

PyShop is a simple e-commerce web application built with Django. The project demonstrates the fundamentals of web development using Python and Django, including product management, user authentication, and an online shopping workflow.

## Features

- Product catalog
- Product detail pages
- Shopping cart
- Admin dashboard
- SQLite database
- Responsive templates
- Django ORM

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite
- Bootstrap

## Installation

Clone the repository:

```bash
git clone https://github.com/abbasmammadoff/pyshop-django.git
```

Go to the project directory:

```bash
cd pyshop-django
```

Install dependencies:

```bash
pip install django
```

Run database migrations:

```bash
python manage.py migrate
```

Start the development server:

```bash
python manage.py runserver
```

Open your browser:

```
http://127.0.0.1:8000/
```
